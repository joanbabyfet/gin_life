## About
使用 gin 开发树莓派桌面小屏应用API接口

## Feature
* 实现websocket向客户端进行数据的推送
* 推送彩云天气信息
* 推送系统信息信息，如CPU、内存、网路
* 推送todo信息

## Usage
```
# 执行
go run main.go
```

## Requires
go v1.19.3
gin v1.9.1
